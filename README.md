hBridge Library
===============

Example library for using the L298 hBridge. Adds features such as 

drive();
leftDrive();
rightDrive();
stop();
coast();
leftStop();
rightStop();

Repository Contents
-------------------

* **/hBridge_libTest/* - Arduino example code for testing the library
